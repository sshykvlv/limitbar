- Автоподхват основного Claude (Keychain `Claude Code-credentials`, read-only) и Codex (`~/.codex/auth.json`, read-only).
- Второй Claude через in-app OAuth PKCE («Add Claude Account…») — куки вставлять не надо. Подтверждено: аккаунты коннектятся.
- «Add Codex Account…» — выбор второго CODEX_HOME (доп. auth.json), read-only.
- Пер-аккаунтные строки: имя (email) + сервис (Claude Code / Codex, нейтральный текст) + два гейджа (5h / неделя).
- Email/тариф подтягиваются через `GET /api/oauth/usage` профиль-эндпоинт (Claude) и JWT id_token (Codex).
- Цветовое кодирование зелёный→жёлтый→красный (>70% / >90% использовано) на гейджах и на иконке.
- Иконка в баре: столбик на аккаунт, высота = израсходовано, белая подложка; тултип = «Claude 86% · …».
- Rename/Remove для всех аккаунтов; удаление builtin-ов «прилипает» (dismissedBuiltins).
- Самообновление с проверкой Developer ID + Team ID `J2Q78NFXZX` + SHA256 (модель Lidless, без авто-swap).
